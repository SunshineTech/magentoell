--------------------------------------------------------------------------------
DESCRIPTION:
--------------------------------------------------------------------------------

This extension manages file uploads. "parent_id" refers to the record ID with
which the uploaded file is associated.

At its initial release, 'parent_id' and 'type' are used as the composite key.

Extension Files:

    - app/etc/modules/SDM_FileUPload.xml
    - app/code/local/SDM/FileUPload/*
