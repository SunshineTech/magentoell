DESCRIPTION:
--------------------------------------------------------------------------------

This extension implements customizations to SDM_NegotiatedProduct module. An
admin can manage a list of products and their negotaited prices at the initial
release.

Extension Files:

    - app/etc/modules/SDM_NegotiatedProduct.xml
    - app/code/local/SD/NegotiatedProduct/*
    - app/design/adminhtml/sdm/ellison/layout/negotiated_product_tab.xml
